# DefSent: Sentence Embeddings using Definition Sentences

arxiv: [https://arxiv.org/abs/2105.04339](https://arxiv.org/abs/2105.04339)

## Installation

```bash
poetry install
```

## Start Mlflow Server

```bash
poetry run mlflow ui
# access http://127.0.0.1:5000
```


## Run Formatter

```bash
poetry run pysen run format
```
